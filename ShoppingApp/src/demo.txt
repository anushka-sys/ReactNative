Good, this is actually simple. Your app already has SignUp → Login → Navigator → HomePage, so you don’t need architecture changes. You only need a tiny auth persistence layer with AsyncStorage and a boolean flag.
Below is the clean minimal implementation exactly matching what you asked.
First understand your current flow (from your code)
Your navigation stack:
GetStartedScreen
LoginScreen
SignUp
DrawerNavigator / TabNavigator → HomePage
Right now login is fake UI only. No persistence. No session.
You want:
Save user id + password during signup
On login → fetch saved data
If match → set flag isLoggedIn = true
Navigate to Home
Next app open → auto login using flag
That’s it.
Step 1 — Install AsyncStorage
Run:

npm install @react-native-async-storage/async-storage
Step 2 — Decide storage keys (important)
Use only 3 keys:

USER_ID
USER_PASS
IS_LOGGED_IN
Keep it stupid simple.

Step 3 — Update SignUp.js (STORE USER)
Open: src/screens/SignUp.js
Find the function where signup button is pressed.
You already collect email & password in state.
Add this import on top:
JavaScript
import AsyncStorage from '@react-native-async-storage/async-storage';
Now add this function inside SignUp screen:
JavaScript
const handleSignUp = async () => {
  try {
    await AsyncStorage.setItem('USER_ID', email);
    await AsyncStorage.setItem('USER_PASS', password);
    await AsyncStorage.setItem('IS_LOGGED_IN', 'true');

    navigation.replace('DrawerNavigator'); // go to home
  } catch (e) {
    console.log('Error saving user', e);
  }
};
Attach it to your signup button:
JavaScript
<TouchableOpacity onPress={handleSignUp}>
Done. Signup now creates account + logs in.

Step 4 — Update LoginScreen.js (CHECK USER)
Open: src/screens/LoginScreen.js
Add import:
JavaScript
import AsyncStorage from '@react-native-async-storage/async-storage';
Add login function:
JavaScript
const handleLogin = async () => {
  try {
    const storedId = await AsyncStorage.getItem('USER_ID');
    const storedPass = await AsyncStorage.getItem('USER_PASS');

    if (email === storedId && password === storedPass) {
      await AsyncStorage.setItem('IS_LOGGED_IN', 'true');
      navigation.replace('DrawerNavigator');
    } else {
      alert('Invalid credentials');
    }
  } catch (e) {
    console.log(e);
  }
};
Attach to login button:
JavaScript
<TouchableOpacity onPress={handleLogin}>
Now login actually validates stored user.


Step 5 — Auto Login when app opens (MOST IMPORTANT)
Open: src/navigation/AppNavigator.js
This is where the app decides first screen.
Add import:
JavaScript
import AsyncStorage from '@react-native-async-storage/async-storage';
import { useEffect, useState } from 'react';
Modify component:
BEFORE (you probably return NavigationContainer directly)
AFTER — add auth check
JavaScript
export default function AppNavigator() {
  const [initialRoute, setInitialRoute] = useState(null);

  useEffect(() => {
    checkLogin();
  }, []);

  const checkLogin = async () => {
    const flag = await AsyncStorage.getItem('IS_LOGGED_IN');
    setInitialRoute(flag === 'true' ? 'DrawerNavigator' : 'GetStartedScreen');
  };

  if (!initialRoute) return null; // wait until check finishes

  return (
    <NavigationContainer>
      <Stack.Navigator screenOptions={{ headerShown: false }} initialRouteName={initialRoute}>
        <Stack.Screen name="GetStartedScreen" component={GetStartedScreen} />
        <Stack.Screen name="LoginScreen" component={LoginScreen} />
        <Stack.Screen name="SignUp" component={SignUp} />
        <Stack.Screen name="DrawerNavigator" component={DrawerNavigator} />
      </Stack.Navigator>
    </NavigationContainer>
  );
}
Now app remembers login state.
Step 6 — Logout (you didn’t ask but you NEED it)
Inside ProfileScreen logout button:
JavaScript
const logout = async () => {
  await AsyncStorage.setItem('IS_LOGGED_IN', 'false');
  navigation.replace('LoginScreen');
};
Final Flow (simple + correct)
App start → check flag
true → Home
false → GetStarted
Signup → save id/pass + flag true → Home
Login → compare → flag true → Home
Logout → flag false → Login
No Redux. No Context. No overengineering. Just storage + flag.
Exactly what you asked.

import AppNavigator from './src/navigation/AppNavigator';
import { PaperProvider } from 'react-native-paper';
import ThemeProvider from './src/context/ThemeContext';
import CartProvider from './src/context/Context';
import { useState, useEffect } from 'react';
import AsyncStorage from '@react-native-async-storage/async-storage';

const App = () => {
  const [initialRoute, setInitialRoute] = useState(null); // null = loading

  useEffect(() => {
    const checkLoginStatus = async () => {
      const isLoggedIn = await AsyncStorage.getItem('IS_LOGGED_IN');
      setInitialRoute(isLoggedIn === 'true' ? 'MainTabs' : 'Login');
    };
    checkLoginStatus();
  }, []);

  if (initialRoute === null) return null; // or a splash/loading screen

  return (
    <ThemeProvider>
      <CartProvider>
        <PaperProvider>
          <AppNavigator initialRoute={initialRoute} />
        </PaperProvider>
      </CartProvider>
    </ThemeProvider>
  );
};

export default App;
